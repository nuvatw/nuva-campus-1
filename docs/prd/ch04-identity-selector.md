| 校園大使卡片 | 展開密碼輸入 Modal |
| 守護者卡片 | 展開密碼輸入 Modal |

## 4.6 組件結構

```typescript
// /app/components/ui/IdentityCard.tsx
interface IdentityCardProps {
  title: string;           // 角色名稱
  subtitle: string;        // 角色說明
  roleKey: 'nunu' | 'ambassador' | 'guardian' | 'other';
  onClick: () => void;
}

// /app/components/ui/PasswordModal.tsx
interface PasswordModalProps {
  isOpen: boolean;
  onClose: () => void;
  roleKey: string;
  onSuccess: (path: string) => void;
}
```

## 4.7 Chain of Thought - 實施步驟

```
1. 建立 IdentityCard 組件
   - 日系簡約卡片設計
   - 細緻的 hover 動畫
   - 支援不同角色配置

2. 建立 PasswordModal 組件
   - 數字鍵盤 UI
   - 支援鍵盤輸入
   - 密碼驗證邏輯（從 DB 讀取）
   - 錯誤震動效果

3. 建立 IdentitySelector 組件
   - 四張卡片的容器
   - 響應式佈局（手機垂直、桌面水平）
   - Modal 狀態管理

4. 重寫 /app/page.tsx
   - 移除現有內容
   - 加入 IdentitySelector
   - 日系極簡背景

5. 添加進場動畫
   - Logo 淡入
   - 卡片依序滑入（stagger）
   - 背景微妙漸變
```
